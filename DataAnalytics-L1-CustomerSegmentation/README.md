
# Customer Segmentation Analysis

## Objective
The objective of this project is to segment customers into different groups based on Annual Income and Spending Score using K-Means Clustering.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
Mall Customers Dataset

## Features
- Data Loading
- Data Exploration
- Missing Value Check
- Statistical Analysis
- Gender Distribution
- Age Distribution
- Annual Income Distribution
- Spending Score Distribution
- Correlation Heatmap
- Feature Scaling
- Elbow Method
- K-Means Clustering
- Customer Segmentation Visualization
- Cluster Summary

## Result
Customers were successfully divided into different segments using K-Means Clustering, helping businesses understand customer behaviour.
