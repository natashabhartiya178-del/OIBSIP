# Retail Sales Data - Exploratory Data Analysis (EDA)

## Internship
OASIS INFOBYTE Internship

## Task
Task 1 - Exploratory Data Analysis (EDA) on Retail Sales Data

## Objective
The objective of this project is to analyze a retail sales dataset and identify sales trends, customer behavior, and business insights using Python.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook (VS Code)

## Dataset
Retail Sales Dataset (CSV)

## Analysis Performed
- Dataset Loading
- Dataset Information
- Shape Analysis
- Missing Value Check
- Descriptive Statistics
- Date Format Conversion
- Monthly Sales Trend
- Quarterly Sales Trend
- Customer Age Distribution
- Gender Distribution
- Product Category Analysis
- Revenue by Product Category
- Correlation Heatmap
- Additional Visualization (Box Plot)

## Business Recommendations
1. Focus on high-revenue product categories.
2. Increase promotional campaigns during low-sales periods.
3. Target customers based on age groups and purchasing behavior.
